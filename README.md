StudyGenie AI

StudyGenie AI is an AI-powered learning assistant that helps students create personalized study plans, solve doubts, generate quizzes, stay motivated, and receive career guidance. Built using Python, Streamlit, Illama,Streamlit and AI technologies, it acts as a 24/7 tutor for students.

Features

- Doubt Solver
- Study Planner
- Quiz Generator
- Motivation Assistant
- Career Guidance

Technologies Used

- Python
- Streamlit
- Ollama
- VS Code

How to Run

1. Install Python
2. Install dependencies
3. Run:
   streamlit run app.py
